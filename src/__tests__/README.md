# Test Suite for macOS Portfolio

Comprehensive unit tests for the macOS Portfolio application.

## Running Tests
- `npm test` - Run all tests
- `npm run test:ui` - Interactive UI
- `npm run test:coverage` - Coverage report

## Test Files
- `store/window.test.js` - Zustand store tests
- `hoc/WindowWrapper.test.jsx` - HOC wrapper tests
- `components/WindowControlls.test.jsx` - Window controls
- `components/Dock.test.jsx` - Dock component
- `windows/Terminal.test.jsx` - Terminal window

Install dependencies: `npm install`
